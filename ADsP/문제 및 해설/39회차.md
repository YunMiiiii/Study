15. 아래 보기에서 설명하는 조직구조는 무엇?
> 전사내에 별도의 분석조직을 보유하고 있으며, 분석조직의 인력을 현업 부서로 배치하여 분석 업무를 수행한다. 전사적 관점에서 분석 과제의 우선순위 선정이 가능하다는 장점이 있다.
> 
> 
> 
<br>1. 집중형 조직 구조 <br>2. 기능 중심 조직 구조 <br>3. 고객 중심 조직 구조<br>4. 분산형 조직 구조
<br><br>**답 : 4**
- **집중형 조직 구조**
    - 조직 내에 별도의 독립적인 분석 전담 조직을 구성함
    - 회사의 모든 분석 업무를 전담 조직에서 담당함
    - 전사적 관점에서 분석 과제의 우선순위를 정해 추진할 수 있음
    - 단점으로 일부 현업 부서와 분석 업무가 중복 또는 이원화될 가능성이 있음
- **기능 중심 조직 구조**
    - 별도의 분석 전담 조직을 구성하지 않고 해당 업무 부서에서 직접 분석하는 형태로 전사적 관점에서 분석이 어렵다는 단점
- **분산형 조직 구조**
    - 별도의 분석 조직에서 각 부서로 인력을 배치하여 분석 업무를 수행하는 구조로 전사적 관점에서 분석 업무에 우선순위를 주어 분석을 실무에 신속히 적용할 수 있는 장점이 있다.

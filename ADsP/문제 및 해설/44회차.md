14. 분석 기획 시 고려사항이 아닌 것은?
    <br> 1. 필요 데이터를 식별하고 수집 가능한지 검토한다.<br> 2. 적절한 활용 방안과 유사 시나리오를 탐색한다. <br>3. 장애요소에 대한 사전 계획을 수립한다. <br>4. 식별된 데이터들에 대한 정합성을 검증한다.

**답 : 4, 식별된 데이터들에 대한 정합성을 검증한다.**

- 가용 데이터에 대한 정합성 검증은 데이터가 수집되고 난 이후에 고려해야할 사항이며, 분석 기획 시 고려해야할 사항으로 적절하지 않다. 분석 기획 시 고려해야할 사항으로는 다음과 같다.
- 가용 데이터 고려 : 분석 목적에 맞는 데이터가 수집 가능한지 파악
- 적절한 활용방안과 유사 시나리오 탐색 : 과거에 유사 분석 수행사례까 있었는 지를 파악하고 기존 분석 사례가 있다면 그 현황을 파악한다.
- 장애요소에 대한 사전 계획 수립 : 분석 수행 시 발생할 수 있는 장애요소에 대한 사전 계획을 수립한다.

답 : 인사이트와지식을 얻는 바텀업 접근 방법이다. 프로토타이핑으로 문제를 더 명확히 파악할 수 있다. 일반적으로 비지도 학습으로 접근한다.
- 프로토타이핑 : 신속하게 해결책이나 모형을 제시함으로써 이를 바탕으로 문제를 좀 더 명확하게 인식하고 필요 데이터를 구체화하는 방법이다.
- 다양한 데이터로부터 인사이트를도출하는 것이 우선이므로 뚜렷한 목적 없이 데이터 탐색을 수행하기 위해 일반적으로 비지도학습을 우선시 수행한다.
- 상향식 접근법에서는 다양한 데이터 간의 상관관계를 탐색하는 데 집중하며, 인과관계는 상대적으로 덜 중요하게 여김

17. 데이터의 가치 측정이 어려운 이유로 적절하지 않은 것은?
    <br>1. 데이터 전문가의 증가로 데이터의 수요가 증가하고 있기 때문이다.<br>2. 데이터는 누가,언제, 어떻게 사용하는 지 알기 어렵기 때문이다. <br>3.과거에는 분석이 불가능했던 것이 분석 기술의 발전으로 분석이 가능해졌기 때문이다. <br> 4. 기존에 존재하지 않았던 새로운 가치를 창출할 수 있기 떄문이다.
**답 : 1**

- 데이터 가치 측정이 어려운 이유로는 다음의 세가지가 있다.
- 데이터 활용 방식 : 데이터 재사용, 재조합 등이 자유로움에 따라 누가, 언제, 어떻게 활용했는지 알 수 없기 때문ㅇ
- 가치 창출 방식 : 빅데이터는 기존에 없던 새로운 가치를 창출함에 따라 그 가치를 사전에 산정하기 어려움
- 분석 기술 발전 : 과거에는 분석할 수 없던 불필요한 데이터들이 분석 기술의 발전으로 인해 높은 가치를 지니게 될지 알 수 없기 때문이다.
